# Team_Project
This program is devoted to saving and maintaining statistics of basketball matches. User can add new teams, players, hold matches between
them and save command and personal statistics. Data is stored in local sqlserver base
Follow this steps in order to prepare an application:

1. Open solution in VS

2. Build solution

3. Open NuGet Package Manager and choose project "BasketballStatistics.Data". Run command "Update-Database". Local sql database
"BasketballStats" will be created and some test data is already loaded in it.

Application is ready for use!
